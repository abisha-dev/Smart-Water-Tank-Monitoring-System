## 📌 Project Overview
This project monitors the water level in a tank using an ultrasonic sensor (HC-SR04).

It provides real-time updates on an LCD display and triggers a buzzer alert when the water level falls below a predefined threshold.

---

## 🛠️ Components Used
- Arduino Uno  
- Ultrasonic Sensor (HC-SR04)  
- 16x2 LCD Display  
- Buzzer  
- Breadboard  
- Jumper Wires  

---

## 🚀 Key Features
✔ Real-time water level monitoring  
✔ Accurate ultrasonic sensing  
✔ LCD display output  
✔ Buzzer alert for low water level  

---

## ⚙️ Working Principle
1. Ultrasonic sensor measures distance to water surface  
2. Distance is converted into water level percentage  
3. LCD displays water level  
4. Buzzer alerts when level is low
